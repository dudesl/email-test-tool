Yet Another Mail Test Suite

A tool for testing mailing mockup build whit node, grunt, sass and using browserSync, mailgun.

For install

$ npm install

This will install all the dependencies. At this point, the suite is ready for test and/or develop.

What can I do whit this?

This suite integrate a set of npm modules that will allow

  • watch files.
  • compile sass.
  • have a live version of the html markup in a local server. whit browser-sync.
  • inline css for mail markup.
  • mailgun for deliver test mails.
